https://www.visualstudio.com/en-us/integrate/api/overview

- Projects
	- Get-VstsProject
- Work
	- Get-VstsWorkItem
	- New-VstsWorkItem
	- Get-VstsWorkItemQuery
- Code	
	- Policy
		- Get-VstsCodePolicy
		- New-VstsCodePolicy
	- Git
		- New-VstsGitRepository
		- Get-VstsGitRepository
